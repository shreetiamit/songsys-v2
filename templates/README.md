# Songsys — Spotify Style Music Recommender

A machine learning based music recommendation system using Spotify audio features.

## Live Demo
https://songsys-v2.onrender.com/

## Tech Stack
- Python (Flask)
- Pandas
- Scikit-learn
- HTML/CSS/JS
- D3.js
- Render (deployment)

## Features
- Music recommendation using cosine similarity
- Spotify-style UI
- Autocomplete search
- Real-time API responses

## ML Approach
- Standardized audio features
- Cosine similarity ranking
- Feature-based recommendation system

## How it works
User selects a song → system finds similar songs using audio feature space → returns top matches.
